# Escape Room Booking

![Tests](https://github.com/john-m-craig/escape-room-booking/actions/workflows/tests.yml/badge.svg)

Free WordPress booking plugin for escape room venues. Up to 2 games, full Stripe payment flow, automated emails and customer self-service.

## Upgrade to Pro

[Escape Room Booking Pro](https://escaperoombookingpro.com) adds unlimited games, promo codes, revenue reports, gamekeepers and more.

## Requirements

- WordPress 6.0+
- PHP 7.4+
- Stripe account
